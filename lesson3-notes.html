Summary for Lesson 4

JavaScript is an additional tool to spice up a webpage and to interact with users in new ways. It allows the website to respond to the user, make decisions, and automate repetitive tasks. The way JavaScript runs is usually within a browser and as a interpreted language.

Here is a list of some of the possible things JavaScript can do:

  • Display messages to users on the webpage, in the browser’s status line, or through alert boxes.
  • Validate form contents and perform automatic calculations (e.g., displaying a running total in an order form).
  • Animate or change images when the mouse moves over them.
  • Create interactive ad banners.
  • Detect the user's browser and perform advanced functions on supported browsers.
  • Detect installed plug-ins and notify the user if any are required.
  • Modify parts of a webpage dynamically without reloading it.
  • Display or interact with data retrieved from a remote server.

Using JavaScript, we can also store data. In the script tag, you can write now = new Date();. Date() is a JavaScript build-in object that gets the date and times. This will then in turn be stored into now.

Example:
<script><br>
  now = new Date();
  localtime = now.toString();
  utctime = now.toGMTString();
  document.write("<p><strong>Local time:</strong> " + localtime + "</p>");
  document.write("<p><strong>UTC time:</strong> " + utctime + "</p>");
</script>

Summary for Lesson 5

As we create web pages, it is important to validate them to make sure they are error free and adhere to the current HTML and CSS standards. This can be done by utilizing an online tool such as https://validator.w3.org

Alternatively we can also use the Developer Tools are are built into the browsers. This can be done by right clicking and selecting inspect. This is useful as it makes debugging easier to identify, especially when the wrong tag is used.

How do you include JavaScript in a web page?

One way to include JavaScript is in the HTML document itself using the script tag. It can also be placed within the body tag, the head tag, withing an HTML tag such as body or form, or in a completely separated file. This file is the .js extension. The best place to put JavaScript is inside the body tag at the very bottom. This ensures that the computer does not take all its time trying to read the JavaScript before HTML and CSS are rendered.
